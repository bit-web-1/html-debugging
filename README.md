# html-debugging

This repo contains a nice, classic HTML file. But there are problems! See if you can find all the bugs in the code and fix them. 

Remember to look for issues with best practice too, not just typos and stuff.

## Instructions:

1. Clone this repo
2. Make a branch for bug fixes
3. Open a Pull Request back to main
4. Use the Inspector in the browser and the features of your editor to find and diagnose code problems
5. As you find and fix the bugs, make a new commit for every bug you fix
6. Make sure you have really good commit messages
7. Remember to push your changes at the end
